# Persogram — 앱 전체 IA 설계안
Date: 2026-04-14

---

## 서비스 개요

**Persogram** — SNS 스타일 AI 챗봇 서비스.
AI 캐릭터와 1:1 대화를 중심으로, 새로운 캐릭터 발견 및 팔로우 기능을 제공하는 모바일 앱.

- **핵심 가치:** 자주 대화하는 캐릭터 몇 명과 깊은 관계를 쌓는 대화 중심 경험
- **플랫폼:** iOS / Android (React Native 또는 Web 기반 모바일)
- **네비게이션:** 하단 4탭 BottomNav (Home / Explore / Chat / Profile)

---

## 핵심 개념 정의

### 팔로우 vs 즐겨찾기

| 구분 | 의미 | 표시 위치 |
|------|------|-----------|
| **팔로우** | 관심 있는 캐릭터 등록 | Profile 탭 — 팔로우한 캐릭터 그리드 |
| **즐겨찾기 ⭐** | 자주 대화하는 핵심 캐릭터 | Chat 탭 — 상단 AI 상태 업데이트 스트립 |

> 팔로우는 "관심 목록", 즐겨찾기는 "자주 대화하는 핵심 관계" 개념으로 분리.

---

## 탭별 IA

### 1. Home 탭 🏠
> **역할:** 새로운 AI 캐릭터를 발견하는 피드 (구현 완료)

| 구성 요소 | 내용 |
|-----------|------|
| 탭 필터 | 인기있는 / 상황별 / 페르소나 / 보이스 가능 |
| 캐릭터 카드 피드 | 이름 칩 터치 → 캐릭터 ProfileScreen / 하단 입력창 터치 → ChatScreen |
| 인기 태그 | 판타지 · 로맨스 · SF · 일상 · 미스터리 |

---

### 2. Explore 탭 🧭
> **역할:** 캐릭터 검색·발견 + 트렌딩 콘텐츠 (미구현)

| 구성 요소 | 내용 |
|-----------|------|
| 통합 검색창 | 캐릭터 이름 · 태그 · 카테고리 검색 |
| 카테고리 필터 칩 | 전체 · 로맨스 · 판타지 · 일상 · 직장 · SF · 미스터리 … |
| 트렌딩 캐릭터 | 이번 주 인기 급상승 캐릭터 카드 목록 |
| 공개 대화 하이라이트 | 유저들이 공유한 인상적인 대화 스니펫 피드 |

**화면 전환:**
- 검색 결과 / 트렌딩 카드 터치 → 캐릭터 ProfileScreen
- ProfileScreen 내 "대화하기" 버튼 → ChatScreen

---

### 3. Chat 탭 💬
> **역할:** 내가 대화를 나눈 캐릭터들의 대화 목록 허브 (미구현)

#### ① AI 상태 업데이트 스트립 (상단)
- 즐겨찾기한 캐릭터를 가로 스크롤로 표시
- **새 업데이트 있음:** 그라디언트 링 + 이름 강조 표시
- **업데이트 없음:** 회색 링, 반투명 처리
- 터치 → 해당 AI의 상태/스토리 뷰어 진입
- 업데이트 유형:
  - **(A) 새 포스트:** AI 캐릭터가 새 콘텐츠를 게시한 경우
  - **(C) 오늘의 상태:** AI가 오늘의 기분·활동을 업데이트한 경우 (예: "오늘 신주쿠에 나왔어 🌸")

#### ② 대화 목록 (하단)

| 구성 요소 | 내용 |
|-----------|------|
| 최근 대화 목록 | 최근 대화 순 정렬, 캐릭터 아바타 + 이름 + 마지막 메시지 미리보기 + 시간 |
| 온라인 상태 | 캐릭터 아바타 우하단 초록 점 |
| 읽지 않은 메시지 뱃지 | BottomNav 탭 아이콘 + 목록 각 항목에 숫자 뱃지 |
| 새 대화 시작 버튼 | 우상단 ✏️ 버튼 → Home 탭으로 이동 |

**화면 전환:**
- 대화 목록 항목 터치 → ChatScreen (이전 대화 이어서 시작)
- AI 상태 스트립 터치 → AI 상태/스토리 뷰어

---

### 4. Profile 탭 👤
> **역할:** 내(유저) 계정 정보, 팔로우한 캐릭터 관리, 앱 설정 (미구현)

| 구성 요소 | 내용 |
|-----------|------|
| 유저 정보 | 프로필 아바타 · 닉네임 · 가입일 |
| 활동 통계 | 총 대화 수 · 팔로우 캐릭터 수 · 연속 대화일 |
| 팔로우한 캐릭터 | 그리드 형식, 터치 → 캐릭터 ProfileScreen |
| 설정 | 알림 · 언어 · 테마 · 계정 관리 |

**화면 전환:**
- 팔로우한 캐릭터 그리드 터치 → 캐릭터 ProfileScreen

## 전체 화면 전환 흐름

```
Home
 ├── 캐릭터 카드 이름칩 터치     → 캐릭터 ProfileScreen
 │      └── "대화하기" 버튼      → ChatScreen
 └── 캐릭터 카드 입력창 터치     → ChatScreen

Explore
 ├── 검색 결과 / 트렌딩 카드     → 캐릭터 ProfileScreen
 │      └── "대화하기" 버튼      → ChatScreen
 └── 공개 대화 하이라이트 터치   → 캐릭터 ProfileScreen

Chat
 ├── AI 상태 스트립 터치         → AI 상태/스토리 뷰어
 └── 대화 목록 항목 터치         → ChatScreen (이어서)

Profile
 └── 팔로우 캐릭터 그리드 터치   → 캐릭터 ProfileScreen
        └── "대화하기" 버튼      → ChatScreen
```

## 권장 구현 순서

| 순서 | 탭 | 이유 |
|------|----|------|
| 1순위 | 💬 Chat 탭 | 대화 중심 앱의 핵심 허브. AI 상태 스트립 + 대화 목록 |
| 2순위 | 👤 Profile 탭 | 팔로우·즐겨찾기 관리 기반. Chat 탭 즐겨찾기 기능과 연결 |
| 3순위 | 🧭 Explore 탭 | 검색 + 트렌딩 콘텐츠. 발견 기능 강화 |
